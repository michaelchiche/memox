## Why

Users need to import text transcription files and transform them into structured, organized insights. Currently, raw transcription files are difficult to navigate and extract value from. This change enables automatic summarization and organization by topic, making transcriptions actionable and searchable.

## What Changes

- Import `.txt` transcription files from the file system
- Parse and process transcription content
- Generate structured summaries via LLM
- Associate each transcription with a topic for organization
- Store processed summaries in a structured format

## Capabilities

### New Capabilities

- `transcription-import`: Import and parse text transcription files
- `llm-summarization`: Generate structured summaries using LLM
- `topic-association`: Link transcriptions and summaries to topics

### Modified Capabilities

## Impact

- New file system import functionality for `.txt` files
- LLM integration for summary generation
- Topic system integration for organizing transcriptions
- Storage layer for persisting processed summaries
